## Frontend - web platform

This is the frontend of the Web Platform application built using Next.js and TypeScript.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS 

Create a .env.local file in the frontend folder:

.env.local:
NEXT_PUBLIC_API_URL=http://localhost:5000

## Execution steps:

1)Clone the Repository:

    git clone repo-url
    cd web-platform/frontend

2)Install Dependencies:

    npm install

3)npm run dev
